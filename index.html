TurtleNav

Turtle Nav is an application for agricultural vehicle GPS navigation in the field using the Google Maps SDK. Works on Android 11.0 and above.

Features

Create markers on map to create fields.

Real-time location using GPS precision.

Manage fields and store them to the app's databse.

Settings fit for agricultural work.

This Android application gives the opportunity for framers to navigate through their field with precision depending on location of the vehicle and the AB Line pattern.

TurtleNav is the ultiamte agriculture solution.
